# Fortress Token Optimizer - GPT Store

GPT (Custom ChatGPT) definition for OpenAI's GPT Store.

## Overview

Fortress Token Optimizer GPT is a ChatGPT configured to automatically optimize your prompts before processing.

## Installation

Available on OpenAI's GPT Store:
- Search for "Fortress Token Optimizer"
- Click "Create"
- Start optimizing immediately

## Features

- Automatic prompt optimization
- Real-time token counting
- Multiple optimization levels
- Transparent workflow (see before/after)

## How It Works

1. Paste your prompt
2. GPT optimizes using Fortress backend
3. Receives optimized prompt + token savings
4. Processes with OpenAI's models
