# Fortress Token Optimizer Website

Marketing site for Fortress Token Optimizer, built with Next.js + TypeScript.

## Features
- Interactive live demos for npm package, GitHub Copilot, VS Code extension, Slack bot, and Claude Desktop
- Realtime "How it works" optimizer with token savings meter
- Usage metrics dashboard with streaming updates
- Install guides for each product integration
- Pricing tiers page

## Development

- npm install
- npm run dev

## Build

- npm run build
- npm run start
